# Attrition RH API

API de prédiction de départ d'employés basée sur un modèle de Machine Learning.

## Endpoints

- `GET /` — Status de l'API
- `GET /health` — Health check
- `POST /predict` — Prédiction de départ
- `GET /docs` — Documentation Swagger
